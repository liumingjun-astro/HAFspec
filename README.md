# HAFspec is a Monte Carlo (MC) simulation code for computing radiative transfer in hot accretion flows (HAFs). It was initially developed to calculate the spectra of hot accretion flows in our disc-corona models [1, 2], but can also be applied to more generalized cases. 

# The basic theory of MC method for spectral calculation of HAFs have been extensively investigated by [3-7]. Here we give a brief description.
